Traceability 101

A practical definition: what to document, how to present it, and how to avoid greenwashing language.

Traceability is only meaningful if it’s specific. We aim to document each component with a trace ID and disclose what we can verify.

What you should be able to see

  • Origin (region / producer, where possible)
  • Mill / processing partner
  • Dye / finish notes
  • Assembly location

We label unknowns as in-progress instead of implying certainty.
